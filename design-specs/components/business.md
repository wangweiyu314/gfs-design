# 广发中后台系统设计规范 — 业务组件

## 1. 图表可视化 (Chart)

### 图表规范

| 项目 | 规范值 | 说明 |
|------|--------|------|
| 标题 | 20px / 28px / 500 / #1D2129 | 所有图表组件顶部标题 |
| 说明 | 14px / 22px / #86909C | 可选的一句话注释 |
| 图表间距 | 标题到图表 16px；图例/注释到图表 16px | 图表内部节奏统一 |
| 轴与网格 | 轴标签 12px / #86909C；网格 #E5E6EB；轴线 #C9CDD4 | 金融指标必须显示单位 |
| 图例 | 标记 12px；文字 12px / 20px；组间距 16px | 折柱混合和双轴图必须区分柱形与线型图例 |
| 业务图表卡片 |- 业务图表卡片：padding 16px；标题 16px / 24px / 500；并列图 gap 16px | 日期、筛选等信息放标题右侧
- 并列图表组合：grid 布局（grid-template-columns + gap 16px），**align-items: start**（顶部对齐），**justify-items: center**（图表和图例在模块内水平居中） |
| 图表容器 | 必须使用独立 class（如 `.chart-wrap`）控制高度，禁止用 `:first-child` 等伪类选择器 | 防止误伤同一布局下的其他模块（如折线图高度与环形图不同） |
| 环图中心与注释 | 中心指标 20px / 28px / 500；标签 12px / 20px；问号 14px | 问号贴字段或中心标签右侧 4px |
| 图形类型 | 环形图、柱形图、双轴图、折线图、面积图、折柱混合图 | 趋势用线，结构用环，对比用柱，量纲不同用双轴 |

### 图表卡片

- 白底 #FFFFFF，默认内边距 20px
- 标准图表卡片约 613 × 390px
- 大型组合卡可扩展到 688px 宽
- 卡片圆角在可视化组件中可为 12px，嵌入业务面板时遵循业务卡片圆角规则

### 环形图

- 常规直径 220px，小卡直径 162px
- 中心指标 20px / 28px / 500 / #1D2129
- **中心文字必须使用固定值，禁止使用模板变量（如 `{d}`）**，防止 hover 时内容变化
- `emphasis` 配置必须显式定义 `label.formatter` 与正常状态一致，确保交互时中心文字保持不变
- **单环形图**：仅使用一个环形展示数据，避免双环/多环嵌套造成视觉混乱
- 多系列数据使用图例区分，而非多环嵌套

### 柱形图

- 单系列柱默认使用 #165DFF 或当前 chart-series-1
- X/Y 轴标签保持 12px
- 标准图形区约 573 × 303px

### 折线图/面积图

- 折线默认使用 chart-series-1，线宽约 2px
- 平滑面积图可使用同色渐变填充

## 2. 筛选区 (Filter Section)

用于数据列表的条件筛选。

### 布局规范

#### 左右结构（默认）

- 容器：flex，wrap，gap 16px，底部对齐
- 筛选项：flex 行，标签与输入框水平排列
- 标签最小宽度：80px，与输入框间距 8px

#### 上下结构（标签较长时）

当筛选标签文字较长（如"申请资格客户编号"、"最早服务开始日期"等）影响排版时，采用上下结构：

- 容器：grid，4 列（`grid-template-columns: repeat(4, 1fr)`），gap 16px 24px
- 筛选项：flex 列，标签在上，输入框在下，gap 8px
- 标签：13px，#939599 (N6)，字重 500，不换行
- 输入框/选择器：宽度 100%，高度 32px
- 日期范围：占 2 列（`grid-column: span 2`），内部两个输入框 + "至" 文本水平排列
- 查询/重置按钮：放在最后一个 grid 单元格内，与最后一项筛选项共用一行，底部对齐

### 筛项规范（上下结构）

- 标签：13px，#939599 (N6)，字重 500
- 输入框边框：1px solid #F0F2F5 (N2)
- 输入框圆角：2px
- 输入框字号：13px，#939599 (N6)
- 聚焦状态：边框 #2A6CDD (Primary)，外发光 rgba(42, 108, 221, 0.1)
- 日期范围：两个输入框 + "至" 文本，各 flex: 1

### 操作按钮

- 查询：主要按钮（Primary）
- 重置：次要按钮（Secondary）
- 按钮位置：与最后一项筛选项在同一 grid 单元格内，避免单独占一行浪费空间
- 按钮文字禁止换行：`white-space: nowrap`

## 3. 工具栏 (Toolbar)

位于筛选区下方、数据表格上方的操作区域。

### 布局规范

- 布局：flex，两端对齐，垂直居中
- 内边距：12px 16px
- 背景：#F7F8FA (N1)
- 边框：底部 1px solid #F0F2F5 (N2)

### 左侧操作区

- 主要操作按钮：新增、导入、导出等
- 按钮间距：12px

### 右侧批量操作区

- 批量信息：13px，#939599 (N6)
- 已选择数量：#2A6CDD (Primary)，字重 600
- 批量删除按钮：次要按钮（默认禁用，有选中项时启用）

## 4. 底部操作栏 (Footer Bar)

页面底部的固定操作栏，常用于表单提交或批量确认。

### 布局规范

- 布局：flex，两端对齐，垂直居中
- 内边距：16px
- 背景：#FFFFFF
- 位置：sticky，底部

### 左侧

- 已选择数量：13px，#939599 (N6)
- 数量值：#2A6CDD (Primary)，字重 600
- 取消选择：文字按钮（Primary 链接样式）

### 右侧

- 操作按钮组：主要按钮 + 次要按钮
- 按钮间距：12px

## 5. 操作按钮组 (Action Buttons)

表格行内的轻量操作按钮。

### 布局规范

- 布局：flex，gap 8px

### 按钮样式

- 内边距：2px 8px
- 字号：12px
- 背景：透明
- 边框：无
- 默认文字色：#2A6CDD (Primary)
- 悬停背景：rgba(42, 108, 221, 0.08)
- 圆角：2px

### 删除按钮

- 文字色：#D12B2B (R4)
- 悬停背景：#FAE6E6 (R1)
