# 广发中后台系统设计规范 — 数据展示组件

## 1. 表格 (Table)

- 边框：#E8E8E8 (N2)
- 表头背景：#F7F8FA（斑马纹表头）/ #F5F5F5 (N1)
- 内容区背景：#FFFFFF（白色）
- 行高：48px（大）/ 40px（中）/ 32px（小）
- 斑马纹：交替 #FAFCFF / #FFFFFF 背景
- 悬停行：#F5F5F5 (N1) 背景
- **数字列字体**：DIN Alternate, DIN, SF Pro Display, Helvetica Neue（金融数据必须使用 DIN 字体）

## 2. 列表 (List)

- 行高：60px（含头像）
- 分割线：底部边框
- 头像尺寸：40px

## 3. 标签 (Tag)

- 圆角：3px
- 字号：12px
- 高度：26px
- 文字左右间距：9px

**标签类型：**

- 默认标签：灰色
- 蓝色标签：#e6f7ff + #2A6CDD
- 成功标签：#f6ffed + #52c41a
- 错误标签：#fff2f0 + #ff4d4f
- 警告标签：#fffbe6 + #faad14

## 4. 树形控件 (Tree)

- 节点图标：文件夹（展开/收起箭头）
- 选中态：蓝色背景 + 白色文字
- 缩进：32px

## 5. 分段控制器 (Segmented)

用于展示多个选项并允许用户选择其中单个选项；切换时关联区域内容变化。

**规格说明：**

- 容器：背景 rgba(240,242,245,.60)，圆角 4px，内边距 4px
- 选中项：白底、2px 圆角、0 2px 8px rgba(2,38,81,.06)，文字 #2A6CDD Medium
- 尺寸：大：外框 48px / 项 40px / 16px 字；小：外框 44px / 项 36px / 14px 字
- 禁用：文字和图标 #C8C9CC，不响应 hover 或点击

**支持类型：**

- 常规（可配合图标使用）
- 禁用
- 图标+文字（左右结构和上下结构）
- 只有图标（需提供可访问名称或 Tooltip）

## 6. 时间轴 (Timeline)

- 节点：12px 圆形
- 连接线：2px 左侧边框
- 颜色：进行中（蓝色）/ 已完成（绿色）/ 未通过（红色）

## 7. 折叠面板 (Collapse)

- 面板标题：14px / 500
- 展开图标：右侧 12px 箭头
- 内容区内边距：16px

## 8. 头像 (Avatar)

- 尺寸：40px / 32px / 24px
- 形状：圆形
- 背景色：使用主色或功能色
- 显示方式：图片 / 文字首字

## 9. 空状态 (Empty)

- 图标：文件夹/文档样式
- 文字：暂无数据
- 可配合操作按钮使用

## 10. 徽标数 (Badge)

- 徽标样式：红色圆点 + 白色数字
- 尺寸：18px × 18px（最小）
- 位置：图标/按钮右上角
- 支持红点模式和数字模式

## 11. 卡片 (Card)

- 卡片间距：16px
- 卡片内边距：16px
- 一行推荐数量：3-4
- 圆角：8px（业务卡片）

**卡片类型：**

- 信息卡片：标题 + 描述 + 操作按钮
- 指标卡片：大数字 + 趋势变化
- 操作卡片：标题 + 描述 + 主要操作按钮

## 12. 描述列表 (Description)

- 标签宽度：120px
- 标签对齐：右对齐
- 数值对齐：左对齐
- 网格间距：16px 24px

## 13. 统计数值 (Statistic)

- 标签：12px / #999
- 数值：32px / 600 / #13161B
- 趋势：12px / #52C41A（上涨）或 #ff4d4f（下跌）

## 14. 图片预览 (ImagePreview)

- 缩略图：120px × 120px
- 圆角：4px
- 悬停：显示遮罩 + 预览文字
- 预览模式：全屏查看

## 15. 水印 (Watermark)

- 内容区域：占满容器
- 水印文字：机密文件
- 角度：-30度
- 透明度：0.15

## 16. 二维码 (QRCode)

- 二维码尺寸：120px × 120px
- 内边距：8px
- 背景：白色

## 17. 代码展示 (Code)

- 背景色：#1e1e1e
- 字体：Monaco / Menlo / monospace
- 字号：13px
- 行高：1.6

## 18. 穿梭框 (Transfer)

- 左右列表：200px 宽度
- 操作按钮：居中排列
- 已选列表支持排序

## 19. 评分 (Rate)

- 星星尺寸：24px（大）/ 20px（小）
- 颜色：#FAAD14
- 支持只读模式

## 20. 骨架屏 (Skeleton)

- 骨架块：灰色背景
- 动画：渐变闪烁效果
- 形状：支持圆形、矩形
