# 广发中后台系统设计规范 — 反馈组件

## 1. 弹窗 (Modal)

### 对话框

- 宽度：480px（小）/ 640px（中）/ 800px（大）
- 圆角：4px
- 阴影：上层投影
- 遮罩：rgba(0, 0, 0, 0.45)

### 抽屉

- 宽度：480px（默认）/ 640px（大）
- 位置：右侧（默认）/ 左侧
- 阴影：中层投影

### 确认框

- 类型：确认 / 信息 / 成功 / 警告 / 错误
- 图标：对应状态图标

## 2. 抽屉 (Drawer)

- 宽度：320px（演示）
- 阴影：-2px 0 8px rgba(0,0,0,0.15)
- 结构：标题栏 + 内容区 + 操作栏

## 3. 文字提示 (Tooltip)

- 位置：上/下/左/右
- 样式：深色背景 + 白色文字
- 延迟显示：300ms

## 4. 加载 (Loading)

- 全局加载：页面级旋转动画
- 局部加载：组件级旋转动画
- 按钮加载：按钮内旋转图标
- 骨架屏：内容占位

## 5. 通知消息 (Notification)

- 位置：右上角
- 类型：成功 / 错误 / 警告 / 信息
- 持续时间：4.5秒
- 可手动关闭

## 6. 消息提示 (Message)

用于全局展示操作反馈信息，自动消失。

**消息类型：**

- 成功消息（绿色背景 + 成功图标）
- 错误消息（红色背景 + 错误图标）
- 警告消息（黄色背景 + 警告图标）
- 信息消息（蓝色背景 + 信息图标）

## 7. 警告提示 (Alert)

用于页面中展示重要的提示信息，不会自动消失。

**警告类型：** 成功 / 错误 / 警告 / 信息

**布局规范：**
- 布局：`display: flex; align-items: center`（图标与文字垂直居中）
- 图标与文字间距：`gap: 8px`
- 内边距：`12px 16px`
- 圆角：`2px`
- 字体大小：13px
- 图标 SVG 必须设置 `flex-shrink: 0`，防止被长文本挤压导致显示不全
- **图标来源**：使用 `icons/status/warning-fill.svg`（16×16 viewBox，fill 路径，无裁剪风险），禁止使用手写 stroke circle 图标
- **SVG 图标 viewBox 裁剪防护**（三重保障）：
  1. **全局规则**：所有 SVG 设置 `overflow: visible`（防止 stroke 被 viewBox 边界裁剪）
  2. **半径约束**：`viewBox="0 0 24 24"` + `stroke-width="2"` 时，`circle` 的 `r` 必须 **≤ 7**（外边缘 = r+1 = **8**，距边界 ≥ **4px**）
  3. **容器防裁剪**：图标容器（`.icon`）设置 `overflow: visible`
  - 公式：`r + stroke-width/2 ≤ viewBox/2 - 4px`
  - 示例：`<circle cx="12" cy="12" r="7" stroke-width="2"/>`

**颜色方案：**
- 信息（info）：背景 `rgba(24, 144, 255, 0.08)`，边框 `1px solid rgba(24, 144, 255, 0.3)`
- 警告（warning）：背景 `#FFF0E0`，边框 `1px solid rgba(250, 140, 22, 0.3)`
- 错误（error）：背景 `#FFF2F0`，边框 `1px solid #FFCCC7`

## 8. 进度条 (Progress)

- 进度条高度：8px
- 圆角：4px
- 颜色：#2A6CDD（主色）
- 支持显示百分比
- 圆形进度圈：支持不同尺寸（40px、60px等）

## 9. 气泡确认框 (Popconfirm)

- 弹出位置：按钮上方
- 箭头：指向触发按钮
- 阴影：中层投影
- 操作按钮：取消 + 确认

## 10. 结果页 (Result)

- 图标：72px 圆形
- 标题：24px / 500
- 描述：14px / #999
- 操作按钮：居中排列
