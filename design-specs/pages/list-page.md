# 广发中后台系统设计规范 — 典型页面：列表页

列表页是金融中后台系统最常见的页面类型，用于展示和管理大量数据记录。

## 页面结构

标准列表页由以下区域自上而下组成：

1. 顶部导航栏（Top Header）
2. 侧边栏导航（Navigation Menu）
3. 多标签导航（MultiTab，可选）
4. 面包屑（Breadcrumb，可选）
5. 页面标题区
6. 标签切换（Tab Switch，可选）
7. 提示信息（Alert，可选）
8. 筛选区（Filter Section）
9. 工具栏（Toolbar）
10. 数据表格（Table）
11. 分页器（Pagination）
12. 底部操作栏（Footer Bar，可选）

## 布局规范

- 页面背景：#F7F8FA (N1)
- 主内容区背景：#FFFFFF
- 内容区内边距：24px
- 内容区最大宽度：自适应
- 各模块间距：16px
- 筛选区与表格区可共用 [卡片](../components/data-display.md#11-卡片-card) 容器

## 各区域规范

### 顶部导航栏

遵循 [顶部导航](../components/navigation.md#5-顶部导航-top-header) 组件规范。

### 侧边栏

遵循 [导航菜单](../components/navigation.md#1-导航菜单) 组件规范。

### 多标签导航

遵循 [多标签导航](../components/navigation.md#6-多标签导航-multitab) 组件规范。

### 面包屑

遵循 [面包屑](../components/navigation.md#2-面包屑-breadcrumb) 组件规范。

### 页面标题区

- 标题：20px / 600 / #262626 (N9)
- 描述（可选）：13px / #8C8C8C (N5)
- 标题区底部间距：16px

### 标签切换

用于在同一页面内切换不同数据视图（如"待申报" / "已申报"）。

- 容器：底部边框 1px solid #E8E8E8 (N2)
- 标签项：内边距 12px 20px，字号 14px，颜色 #595959 (N6)
- 选中态：底部 2px 边框 #2A6CDD (Primary)，文字 #2A6CDD，字重 500
- 徽标数：min-width 18px，高度 18px，圆角 9px，背景 #D12B2B (R4)，白色文字，11px

### 提示信息

用于筛选区上方展示重要业务提示。

- 类型：信息（info）/ 警告（warning）
- 信息提示：背景 rgba(24, 144, 255, 0.08)，边框 1px solid rgba(24, 144, 255, 0.3)
- 警告提示：背景 #FFF0E0 (O1)，边框 1px solid rgba(250, 140, 22, 0.3)
- 内边距：12px 16px
- 圆角：2px
- 图标尺寸：20px
- 标题：13px / 500
- 内容：13px / 1.6 行高

### 筛选区

遵循 [筛选区](../components/business.md#2-筛选区-filter-section) 组件规范。

**扩展规则：**
- 筛选条件超过一行时，默认折叠，显示"展开/收起"按钮
- 常用筛选条件可记忆用户上次选择
- 筛选区可使用 [卡片](../components/data-display.md#11-卡片-card) 包裹，背景 #FFFFFF，内边距 16px
- 日期范围：两个 130px 输入框 + "至" 文本
- 聚焦状态：边框 #2A6CDD，外发光 rgba(42, 108, 221, 0.1)

### 工具栏

遵循 [工具栏](../components/business.md#3-工具栏-toolbar) 组件规范。

**扩展规则：**
- 左侧主要操作：新增、导入、导出等业务操作按钮
- 右侧批量操作：批量删除、全部删除等（默认禁用，有选中项时启用）

### 数据表格

遵循 [表格](../components/data-display.md#1-表格-table) 组件规范。

**列表页扩展规则：**
- 表格可使用 [卡片](../components/data-display.md#11-卡片-card) 包裹，与筛选区共用卡片时去除上边框
- 支持斑马纹（交替 #FAFCFF / #FFFFFF）
- 支持行内复选框，表头支持全选
- 行内操作使用 [操作按钮组](../components/business.md#5-操作按钮组-action-buttons)
- 状态列使用 [标签](../components/data-display.md#3-标签-tag) 组件（如买入/卖出、待申报/已申报）
- 表头背景：#F7F8FA，字重 600，颜色 #434343 (N7)
- 表格字号：13px

**数字列字体检查清单（开发自检）：**
- [ ] 编号类列（服务编号、申请编号、客户编号）表头与数据列均添加 `.number-col`
- [ ] 金额类列（补偿金额、申请金额）表头与数据列均添加 `.number-col`
- [ ] 日期类列（开始日期、结束日期、申请日期）表头与数据列均添加 `.number-col`
- [ ] 期数/计数类列（如 2/4）数据列添加 `.number-col`
- [ ] CSS 选择器已覆盖嵌套结构：`.number-col, .number-col div:first-child`
- [ ] 字体栈顺序正确：`DIN Alternate` > `DIN` > `SF Pro Display`

### 分页器

遵循 [分页器](../components/navigation.md#10-分页器-pagination) 组件规范。

**布局规则：**
- 分页器位于表格右下角
- 与表格间距：16px
- 支持每页条数选择（10/20/50）
- 支持快速跳转至指定页

### 底部操作栏

遵循 [底部操作栏](../components/business.md#4-底部操作栏-footer-bar) 组件规范。

**触发条件：** 仅当表格启用批量选择且用户选中至少一行时显示。

**按钮配置：**
- 左侧：已选择数量（13px，#2A6CDD，字重 600）+ 取消选择文字按钮
- 右侧：根据业务场景配置，如"暂存申报"（次要按钮）+ "确认申报"（主要按钮）

## 交互规范

- 筛选条件变更后点击查询按钮触发查询
- 表格行悬停显示操作按钮组
- 点击行可进入详情页（或在新标签页打开）
- 分页切换保留当前筛选条件
- 标签切换保留筛选条件
- 复选框选中/取消实时更新底部操作栏已选择数量
