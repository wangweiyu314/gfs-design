# 广发中后台系统设计规范 — 间距与布局

## 1. 间距系统

采用 8px 基准网格系统，4px 为最小单位：

| Token | 数值 | CSS 变量 | 用途 |
|-------|------|----------|------|
| space-0 | 0px | --space-0: 0px | 无间距 |
| space-1 | 4px | --space-1: 4px | 紧凑间距（图标与文字间） |
| space-2 | 8px | --space-2: 8px | 小间距（内边距紧凑） |
| space-3 | 12px | --space-3: 12px | 中间距（输入框内边距） |
| space-4 | 16px | --space-4: 16px | 默认间距（卡片内边距） |
| space-5 | 20px | --space-5: 20px | 中间距 |
| space-6 | 24px | --space-6: 24px | 大间距（表单项间距） |
| space-8 | 32px | --space-8: 32px | 区块间距 |
| space-10 | 40px | --space-10: 40px | 大区块间距 |
| space-12 | 48px | --space-12: 48px | 页面间距 |
| space-16 | 64px | --space-16: 64px | 大页面间距 |

## 2. 布局规范

**页面布局**

- 页面宽度：1440px（标准桌面端）
- 页面高度：自适应，最小 900px
- 内容区宽度：自适应，最大 1200px
- 侧边栏宽度：200px（展开）/ 64px（折叠）

**栅格系统**

- 栅格列数：24列
- 列宽：自适应
- 列间距：16px（默认）/ 24px（宽松）
- 边距：24px（默认）/ 32px（宽松）

**卡片布局**

- 卡片内边距：16px（紧凑）/ 24px（默认）
- 卡片圆角：4px（默认，边角-2）
- 卡片阴影：0 2px 8px rgba(0, 0, 0, 0.08)

## 3. 响应式断点

| 断点 | 宽度 | 设备类型 |
|------|------|----------|
| xs | < 576px | 手机 |
| sm | ≥ 576px | 大手机 |
| md | ≥ 768px | 平板 |
| lg | ≥ 992px | 小桌面 |
| xl | ≥ 1200px | 桌面 |
| xxl | ≥ 1440px | 大桌面 |
