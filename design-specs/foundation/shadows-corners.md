# 广发中后台系统设计规范 — 阴影与边角

## 1. 阴影系统

系统定义了三层阴影，每层包含三个方向（下方、左侧、右侧）：

### 下层投影（Low Shadow）

| 方向 | CSS 值 | 使用场景 |
|------|--------|----------|
| 下方 | 0px 2px 8px rgba(0, 0, 0, 0.10) | 组件 hover 状态，例如表格行和树拖动 |
| 左侧 | -4px 0px 8px rgba(0, 0, 0, 0.10) | 右侧面板悬停 |
| 右侧 | 4px 0px 8px rgba(0, 0, 0, 0.10) | 左侧面板悬停 |

### 中层投影（Medium Shadow）

| 方向 | CSS 值 | 使用场景 |
|------|--------|----------|
| 下方 | 0px 6px 16px rgba(0, 0, 0, 0.12) | 下拉组件，例如下拉菜单 / 气泡确认框 / 选择器 |
| 左侧 | -8px 0px 16px rgba(0, 0, 0, 0.12) | 右侧抽屉 |
| 右侧 | 8px 0px 16px rgba(0, 0, 0, 0.12) | 左侧抽屉 |

### 上层投影（High Shadow）

| 方向 | CSS 值 | 使用场景 |
|------|--------|----------|
| 下方 | 0px 12px 32px rgba(0, 0, 0, 0.14) | 警示或弹窗组件，例如全局提示 / 消息通知 |
| 左侧 | -12px 0px 32px rgba(0, 0, 0, 0.14) | 全屏抽屉（右侧） |
| 右侧 | 12px 0px 32px rgba(0, 0, 0, 0.14) | 全屏抽屉（左侧） |

## 2. 边角系统

系统仅使用两种圆角值：

| 名称 | 数值 | 使用场景 |
|------|------|----------|
| 边角-1 | 2px | 按钮 |
| 边角-2 | 4px | 提示框、卡片、输入框、下拉面板 |

**使用规则：** 按钮统一使用 2px 圆角；输入框、卡片、弹窗、下拉面板统一使用 4px 圆角；不使用更大的圆角值。
