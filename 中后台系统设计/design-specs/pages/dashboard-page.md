# 广发中后台系统设计规范 — 典型页面：仪表盘页

仪表盘页用于展示关键业务指标和数据可视化，是系统首页或管理驾驶舱的核心页面类型。

## 页面结构

标准仪表盘页由以下区域自上而下组成：

1. 顶部导航栏（Top Header）
2. 侧边栏导航（Navigation Menu）
3. 搜索区（Search，可选）
4. 业务概览区块（Business Overview）
   - 区块标题（Block Header）
   - 筛选表单
   - KPI 标签卡片（KPI Tab Cards）
5. 图表可视化区（Chart Cards）
6. 客户/数据概览区（Data Overview，可选）
   - 维度切换标签
   - 数据表格卡片（Table Cards）

## 布局规范

- 页面背景：#FFFFFF (白色)
- 内容区内边距：24px
- 区块间距：24px
- 卡片间距：16px
- 图表卡片栅格：2列等宽（列间距 16px）

## 各区域规范

### 顶部导航栏

遵循 [顶部导航](../components/navigation.md#5-顶部导航-top-header) 组件规范。

### 侧边栏

遵循 [导航菜单](../components/navigation.md#1-导航菜单) 组件规范。

### 搜索区

用于快速搜索标的或关键数据。

- 搜索框容器：flex，gap 8px
- 输入框：高度 32px，内边距 8px 12px，圆角 4px，边框 1px solid #E8E8E8
- 搜索图标：16px，置于输入框左侧
- 搜索按钮：主要按钮，文字如"券一下"

### 业务概览区块

#### 区块标题

- 容器：flex，align-items center，gap 8px
- 左侧色条：4px 宽，#2A6CDD，高度 16px
- 标题：16px / 500 / #13161B (N9)

#### 筛选表单

- 布局：flex，wrap，gap 16px，底部对齐
- 时间范围：两个 [日期输入框](../components/data-entry.md#5-日期选择器-datepicker) + "至" 文本
- 下拉选择：宽度 200px，高度 32px，边框 1px solid #F0F2F5
- 操作按钮：查询（主要按钮）+ 重置（次要按钮）

#### KPI 标签卡片

用于展示核心业务指标，支持切换不同业务类型（如"融券" / "转融券"）。

- 容器：flex，gap 16px
- 卡片宽度：自适应，约 50%
- 卡片背景：#FFFFFF
- 卡片内边距：20px
- 卡片圆角：4px

**激活态卡片：**

- 标签文字：14px / #939599 (N6)
- 主数值：24px / 600 / #13161B (N9)
- 百分比：14px / 500
- 副标题：12px / #8C8C8C (N5)
- 底部网格：3 行数据，12px / #939599
- 趋势文字（白底卡片）：上涨红色 #F5222D / 下跌绿色 #52C41A
- 趋势文字（蓝底/渐变卡片）：上涨浅粉 #FFB3B3 / 下跌浅青 #69E0D8（保证深色背景可读性）
- 趋势图标：14px，使用 ChevronUp / ChevronDown

**非激活态卡片：**

- 标签文字：14px / #939599 (N6)
- 主数值：24px / 600 / #13161B (N9)
- 状态标签：背景 #FAAD14（警告黄），文字 #FFFFFF（白色），12px，padding 2px 8px，圆角 2px（如"待更新"）
- 底部网格同激活态

### 图表可视化区

使用 [图表可视化](../components/business.md#1-图表可视化-chart) 组件。

**布局规则：**

- 图表区块使用 [卡片](../components/data-display.md#11-卡片-card) 包裹
- 标准图表：两列排列
- 全宽图表：占满整行（100% 宽度）
- 并列图表：同一卡片内左右排列（如环形图组合）

**图表卡片头部：**

- 标题：16px / 500 / #1D2129
- 副标题：14px / #86909C
- 分段控制器：用于切换图表维度（如"按期末余额" / "按日均余额"）

**图表类型配置：**

| 图表类型 | 用途 | 规格 |
|----------|------|------|
| 折柱混合图 | 趋势对比（市场规模 + 市占率） | 双 Y 轴，折线宽 2px，柱形默认 #2A6CDD |
| 环形图 | 业务构成占比 | 直径 220px，中心指标 20px / 500 |
| 双环形图 | 多维度业务构成（如按板块/按客户类型） | **左右并列两个独立单环形图**，每个环形图有自己的图例和数据，禁止双环嵌套 |
| 柱形图 | 经营情况对比 | 单系列默认 #2A6CDD，X/Y 轴标签 12px |
| 组合图 | 分公司数据 | 左侧环形图（中心显示数值+标签）+ 右侧双 Y 轴折线图（两条线+图例），各占 50% |

### 客户/数据概览区

#### 维度切换

- 容器：flex，gap 8px
- 标签按钮：内边距 6px 16px，圆角 2px，背景 #FFFFFF，边框 1px solid #E8E8E8
- 选中态：背景 #2A6CDD，文字白色，无边框

#### 数据表格卡片

- 卡片头部：flex，两端对齐
  - 标题：16px / 500 / #13161B
  - 工具区：[分段控制器](../components/data-display.md#5-分段控制器-segmented)（前五/后五）+ "更多"链接
- 表格：13px，表头背景 #F7F8FA，字重 600
- 趋势列：上涨红色 / 下跌绿色，数值前带 +/- 符号
- 支持横向滚动（`overflow-x: auto`）

## 交互规范

- KPI 卡片点击切换业务类型，关联图表数据联动更新
- 图表支持悬停查看详细数据 Tooltip
- 表格"更多"链接跳转至完整列表页
- 分段控制器切换保留当前筛选条件
- 数据支持自动刷新（默认 5 分钟）
